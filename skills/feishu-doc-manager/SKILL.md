# Feishu Doc Manager - 飞书文档管理专家

## 概述

这是一个全面的飞书文档管理技能，帮助您高效处理飞书文档、知识库和多维表格的所有操作。支持从各种格式（文本、Markdown、PDF）导入内容到飞书文档，提供PDF OCR识别和中英文翻译功能，以及合同协议模板的管理和快速生成。

## 核心能力

### 1. 文档基础操作

#### 创建文档
```markdown
**使用场景：**
- "帮我创建一个新的会议纪要文档"
- "在飞书中创建一个产品需求文档"

**操作步骤：**
1. 确定文档类型和标题
2. 使用 `feishu_doc` 工具创建文档
3. 添加基础内容结构
4. 返回文档链接给用户
```

#### 编辑文档
```markdown
**使用场景：**
- "帮我更新这个飞书文档的内容"
- "在文档末尾添加新的章节"

**操作步骤：**
1. 获取文档 token（从 URL 中提取）
2. 读取现有内容（如需）
3. 使用 `feishu_doc` 工具进行编辑
4. 确认修改完成
```

#### 文档读取
```markdown
**使用场景：**
- "帮我读取这个飞书文档的内容"
- "把这个文档的内容转成 Markdown"

**操作步骤：**
1. 从 URL 提取 doc_token
2. 使用 `feishu_doc` 的 `read` 操作
3. 根据需要转换格式
```

### 2. 内容导入与转换

#### 文本/Markdown → 飞书文档
```markdown
**使用场景：**
- "把这段文本转成飞书文档"
- "这个 Markdown 文件帮我导入到飞书"
- "我有一段内容，帮我创建一个飞书文档"

**操作步骤：**
1. 接收用户提供的文本或文件内容
2. 分析内容结构（标题、段落、列表等）
3. 转换为飞书文档支持的格式
4. 使用 `feishu_doc` 创建文档
5. 应用适当的格式和样式
6. 返回文档链接

**格式转换要点：**
- Markdown 标题 → 飞书文档标题块
- Markdown 列表 → 飞书文档列表块
- 代码块 → 飞书文档代码块
- 图片 → 上传并插入（如需）
- 链接 → 保持链接格式
```

#### PDF → Markdown（OCR识别）
```markdown
**使用场景：**
- "帮我把这个 PDF 转成可编辑的文本"
- "识别这个 PDF 的内容"

**操作步骤：**
1. 获取 PDF 文件（用户上传或提供路径）
2. 使用 OCR 工具提取文本内容
   - 优先使用 `pdfplumber` 或 `PyPDF2`
   - 如需 OCR，使用 `pytesseract` 或 `easyocr`
3. 整理提取的内容，保持结构
4. 转换为 Markdown 格式
5. 返回给用户或继续处理

**注意：**
- 扫描版 PDF 需要 OCR
- 文字版 PDF 可以直接提取
- 复杂布局可能需要手动调整
```

#### 英文PDF → 中文翻译
```markdown
**使用场景：**
- "把这个英文 PDF 翻译成中文"
- "帮我翻译这篇外文文章"

**操作步骤：**
1. 先执行 PDF → Markdown 转换
2. 识别内容语言
3. 进行翻译（使用可用的翻译工具或模型能力）
4. 保持原文结构和格式
5. 生成中英对照或纯中文版本
6. 可选择创建飞书文档保存翻译结果

**翻译要点：**
- 保持专业术语准确
- 注意上下文和语境
- 技术文档要保持专业性
- 可以分段翻译并验证
```

### 3. 合同协议模板管理

#### 模板库管理
```markdown
**使用场景：**
- "帮我管理合同模板"
- "保存这个 APA 协议模板"
- "列出所有可用的合同模板"

**模板存储位置：**
- 优先使用飞书知识库（Wiki）存储模板
- 或使用飞书文档文件夹组织
- 使用多维表格（Bitable）管理模板元数据

**模板元数据应该包含：**
- 模板名称
- 模板类型（APA协议、授权协议、合作协议等）
- 适用场景
- 创建日期
- 最后更新日期
- 版本号
- 关联文档链接

**操作步骤：**
1. 创建模板文档
2. 在 Bitable 中记录模板信息
3. 设置适当的分类和标签
4. 建立模板索引文档
```

#### 快速生成定制化合同
```markdown
**使用场景：**
- "根据这个客户信息生成一份合作协议"
- "帮我做一份适用于 A 公司的 APA 协议"
- "快速生成一份授权合同"

**操作步骤：**
1. 收集客户信息和需求
   - 客户名称
   - 合作方信息
   - 特殊条款要求
   - 合同期限
   - 金额/费用信息
   - 其他定制化需求

2. 选择合适的模板
   - 根据合同类型匹配模板
   - 考虑行业和场景

3. 填充变量
   - 公司名称替换
   - 日期填充
   - 金额填写
   - 特殊条款插入

4. 生成最终文档
   - 使用 `feishu_doc` 创建新文档
   - 应用模板内容
   - 替换所有变量
   - 格式检查和调整

5. 审核和确认
   - 检查关键信息
   - 确认条款完整性
   - 返回文档链接给用户

**常用变量占位符：**
- `{{公司名称}}` - 甲方/乙方公司全称
- `{{公司简称}}` - 公司简称
- `{{签署日期}}` - 合同签署日期
- `{{合同期限}}` - 合同有效期
- `{{金额}}` - 合同金额
- `{{联系人}}` - 联系人姓名
- `{{联系电话}}` - 联系电话
- `{{联系邮箱}}` - 联系邮箱
- `{{地址}}` - 公司地址
```

### 4. 多维表格（Bitable）管理

#### 创建和管理表格
```markdown
**使用场景：**
- "帮我创建一个合同模板管理表格"
- "在这个 Bitable 中添加一条记录"
- "查看这个表格的数据"

**常用操作：**
1. 创建 Bitable 应用
   - 使用 `feishu_bitable_create_app`
   - 设置应用名称

2. 管理表格字段
   - 使用 `feishu_bitable_list_fields` 查看字段
   - 使用 `feishu_bitable_create_field` 添加字段

3. 操作记录
   - 使用 `feishu_bitable_list_records` 列出记录
   - 使用 `feishu_bitable_create_record` 创建记录
   - 使用 `feishu_bitable_update_record` 更新记录
   - 使用 `feishu_bitable_get_record` 获取单条记录

**合同模板管理表格建议字段：**
- 模板名称（文本）
- 模板类型（单选：APA协议、授权协议、合作协议等）
- 适用场景（文本）
- 文档链接（URL）
- 创建日期（日期）
- 最后更新日期（日期）
- 版本号（文本）
- 状态（单选：在用、停用、草稿）
- 备注（多行文本）
```

### 5. 知识库（Wiki）管理

#### 空间和节点管理
```markdown
**使用场景：**
- "帮我查看有哪些知识库空间"
- "在这个空间下创建一个新页面"
- "整理知识库结构"

**操作步骤：**
1. 列出知识库空间
   - 使用 `feishu_wiki` 的 `spaces` 操作

2. 管理节点
   - 使用 `feishu_wiki` 的 `nodes` 操作列出节点
   - 使用 `create` 操作创建新节点
   - 使用 `move` 操作移动节点
   - 使用 `rename` 操作重命名节点

**注意：**
- 需要先在知识库设置中添加机器人为成员
- 确保有足够的权限进行操作
```

### 6. 批量操作和格式规范

#### 批量文档操作
```markdown
**使用场景：**
- "把这些文档都移动到同一个文件夹"
- "批量更新这些文档的标题格式"
- "导出多个文档的内容"

**操作要点：**
1. 先列出需要操作的文档
2. 制定操作计划
3. 逐个执行操作（注意速率限制）
4. 记录操作结果
5. 验证操作完成
```

#### 格式规范化
```markdown
**使用场景：**
- "帮我规范这个文档的格式"
- "统一这些文档的标题样式"
- "调整文档的排版"

**规范要点：**
- 标题层级统一（H1, H2, H3...）
- 段落间距一致
- 列表格式规范
- 代码块样式统一
- 图片大小和位置调整
- 链接格式统一
```

```
用户请求
    │
    ├── 是创建/编辑/读取文档？ → 文档基础操作
    │
    ├── 是内容导入/转换？
    │   ├── 文本/Markdown → 飞书文档 → 内容导入流程
    │   ├── PDF → Markdown → PDF转换流程
    │   └── PDF翻译 → 翻译流程
    │
    ├── 是合同模板相关？
    │   ├── 管理模板 → 模板库管理
    │   └── 生成合同 → 合同生成流程
    │
    ├── 是 Bitable 相关？ → 多维表格管理
    │
    ├── 是 Wiki 相关？ → 知识库管理
    │
    └── 其他？ → 分析需求，选择合适的工具组合
```

## 常用工具清单

### Feishu 文档工具
- `feishu_doc` - 文档的读、写、追加、插入、创建等
- `feishu_wiki` - 知识库空间和节点管理
- `feishu_drive` - 云存储文件管理
- `feishu_bitable_*` - 多维表格操作系列
- `feishu_chat` - 聊天操作
- `feishu_app_scopes` - 查看权限

### 内容处理工具
- `pdfplumber` / `PyPDF2` - PDF 文本提取
- `pytesseract` / `easyocr` - OCR 识别
- Markdown 解析和生成
