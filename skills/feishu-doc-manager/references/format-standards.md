# 文档格式规范手册

## 标题层级规范

### H1 - 一级标题
**用途：** 文档主标题
**格式：**
- 字体：黑体，字号 22-24pt
- 居中或左对齐
- 段前间距：18-24pt
- 段后间距：12-18pt
- 加粗

**Markdown 示例：**
```markdown
# 文档主标题
```

### H2 - 二级标题
**用途：** 主要章节标题
**格式：**
- 字体：黑体，字号 18-20pt
- 左对齐
- 段前间距：12-18pt
- 段后间距：6-12pt
- 加粗
- 编号：1.、2.、3. 或 一、二、三、

**Markdown 示例：**
```markdown
## 1. 主要章节
```

### H3 - 三级标题
**用途：** 子章节标题
**格式：**
- 字体：黑体或楷体，字号 16-18pt
- 左对齐
- 段前间距：6-12pt
- 段后间距：3-6pt
- 加粗
- 编号：1.1、1.2 或 （一）、（二）

**Markdown 示例：**
```markdown
### 1.1 子章节
```

### H4 - 四级标题
**用途：** 小节标题
**格式：**
- 字体：楷体或宋体，字号 14-16pt
- 左对齐
- 段前间距：3-6pt
- 段后间距：3pt
- 加粗（可选）
- 编号：1.1.1 或 1.

**Markdown 示例：**
```markdown
#### 1.1.1 小节
```

## 正文格式规范

### 段落
**格式：**
- 字体：宋体，字号 12-14pt
- 行间距：1.5 倍或 1.6 倍
- 段前间距：0-3pt
- 段后间距：3-6pt
- 首行缩进：2 字符
- 对齐方式：两端对齐

**Markdown 示例：**
```markdown
这是一个正文段落。段落应该有适当的行间距和段间距，确保阅读舒适。首行缩进 2 字符，采用两端对齐。
```

### 列表

#### 有序列表
**用途：** 步骤、序列等有顺序的内容
**格式：**
- 编号：1.、2.、3.
- 缩进：与正文首行缩进对齐
- 层级：最多 3-4 层

**Markdown 示例：**
```markdown
1. 第一步
2. 第二步
   1. 子步骤 2.1
   2. 子步骤 2.2
3. 第三步
```

#### 无序列表
**用途：** 要点、项目等无顺序的内容
**格式：**
- 符号：•、-、○、□
- 缩进：与正文首行缩进对齐
- 层级：最多 3-4 层

**Markdown 示例：**
```markdown
- 要点一
- 要点二
  - 子要点 A
  - 子要点 B
- 要点三
```

### 代码块
**格式：**
- 字体：等宽字体（Consolas、Courier New）
- 字号：10-12pt
- 背景色：浅灰色
- 缩进：4 空格
- 边框：可选

**Markdown 示例：**
```python
def hello_world():
    print("Hello, World!")
```

### 引用块
**格式：**
- 左边框：2-3pt 彩色线条
- 左缩进：1-2 字符
- 字体：斜体（可选）
- 背景色：浅灰色（可选）

**Markdown 示例：**
```markdown
> 这是一段引用文字。
> 可以有多行。
```

## 表格格式规范

### 表格结构
**格式：**
- 表头：加粗，背景色区分
- 边框：清晰的网格线
- 对齐：数字右对齐，文本左对齐，标题居中
- 间距：适当的单元格内边距

**Markdown 示例：**
```markdown
| 列名1 | 列名2 | 列名3 |
|-------|-------|-------|
| 数据1 | 数据2 | 数据3 |
| 数据4 | 数据5 | 数据6 |
```

### 表格对齐
```markdown
| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 文本   |   标题   |  123   |
| 内容   |   数值   |  456   |
```

## 图片格式规范

### 图片插入
**格式：**
- 对齐：居中
- 大小：宽度不超过页面宽度的 80%
- 清晰度：确保清晰可读
- 格式：PNG、JPG、SVG 等

**Markdown 示例：**
```markdown
![图片描述](图片链接)
```

### 图片标题
**格式：**
- 位置：图片下方
- 格式："图 X-X 标题内容"
- 对齐：居中
- 字号：比正文小 1-2pt

**Markdown 示例：**
```markdown
![系统架构图](architecture.png)

图 1-1 系统架构图
```

## 链接格式规范

### 文本链接
**格式：**
- 颜色：蓝色
- 下划线：有
- 提示：鼠标悬停显示链接地址

**Markdown 示例：**
```markdown
[链接文本](https://example.com)
```

### 参考链接
```markdown
[链接文本][ref1]

[ref1]: https://example.com
```

## 合同文档特殊规范

### 合同编号
**格式：** `{{公司缩写}}-{{合同类型}}-{{年份}}-{{序号}}`
**示例：** `ABC-APA-2024-001`

### 签署页格式
- 双方信息：左右分栏
- 盖章位置：预留足够空间
- 日期位置：盖章下方
- 联系人信息：可选

### 页码
- 位置：页脚居中
- 格式："第 X 页 共 Y 页"
- 起始页：正文第一页
